# Reference Materials

This folder contains reference materials from the original CotEditor macOS project. These are **not part of the iOS app codebase** but are kept for inspiration and reference.

## Contents

### Themes/
Original CotEditor theme definitions in property list format. These can be referenced when creating new themes for the iOS app.

Each theme contains:
- `text` - Default text color
- `background` - Editor background color
- `insertionPoint` - Cursor color
- `selection` - Selection highlight color
- `invisibles` - Color for invisible characters
- `syntax` colors for: keywords, commands, types, attributes, variables, values, numbers, strings, characters, comments
